EduStore

Inventory made simple

EduStore is an extensive and secure inventory system made for the modern school district

Easily Track the Number of Items

EduStore lets you easily see the current number of all items in inventory and quickly add more items when new shipments are received. Sort items by name and include purchase order numbers for better organization.

Manage User Access and Control

Create and manage user and administrator accounts, allowing you to limit which users have access to remove, add, or edit items in inventory.
